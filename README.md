# Trivia
Trivia game for multiple users over a network made for fun and learning purposes. 

## Usage
1. Run server.py
2. Run client.py for every user who wishes to play the game.
3. Log in with one or more of the following users:
    *user1: pass1
    *user2: pass2
    *user3: pass3
4. Play the game :)

## Resetting Database
`python server.py -r`
